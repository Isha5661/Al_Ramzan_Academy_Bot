# 🤖 Al Ramzan Academy Bot

Welcome to the **Al Ramzan Academy Bot**! This is a professional AI-powered chatbot designed specifically to assist students with programming, Python concepts, and computer science learning.

### 🚀 Live Demo
You can try out the live chatbot here: 
👉 **[alramzanacademybot.streamlit.app](https://alramzanacademybot.streamlit.app/)**

---

## ✨ Features
- **Instant Coding Help:** Answers queries related to Python, Object-Oriented Programming (OOP), and general computer science.
- **Smart Learning Companion:** Helps students of Al Ramzan Academy clear their doubts anytime.
- **Secure Backend:** Uses Streamlit Secrets to manage API keys securely, protecting against unauthorized leaks.
- **Persistent Chat History:** Maintains the conversation context within the session.

---

## 🛠️ Tech Stack & Tools Used
- **Language:** Python 🐍
- **Framework:** Streamlit (For Web UI)
- **AI Model:** Google Gemini Pro API (via `google-generativeai` SDK)
- **Deployment:** Streamlit Community Cloud
- **Version Control:** GitHub
